# 🍳 CuteOrder Web — 网页版

gogo & putato 专属点菜 Web 应用 💕

## 使用方式

### 直接打开（开发时）
在浏览器中直接打开 `index.html` 即可使用。

### 部署上线（免费）
部署到 **Cloudflare Pages** 或 **Vercel**，你和 putato 就能在手机上通过链接访问了。

#### 部署到 Cloudflare Pages（推荐，永久免费）
1. 注册 [Cloudflare](https://dash.cloudflare.com) 账号
2. 进入 **Workers 和 Pages** → **Pages** → **创建项目**
3. 连接你的 GitHub/GitLab 仓库，或直接上传 `cute-order-web/` 目录
4. 框架预设选 **无**，其余保持默认 → **保存并部署**
5. 部署完成后会得到一个 `xxx.pages.dev` 的链接
6. 把链接发给 putato，手机浏览器打开就能用 🎉

#### 部署到 Vercel
1. 注册 [Vercel](https://vercel.com) 账号
2. 安装 Vercel CLI：`npm i -g vercel`
3. 在 `cute-order-web/` 目录下运行 `vercel`
4. 部署完成后会得到一个链接

### 添加到手机桌面（像原生 App）
iOS Safari：
1. 打开链接 → 底部分享按钮
2. 向下滑 → **「添加到主屏幕」**
3. 改名 → 添加 ✅

Android Chrome：
1. 打开链接 → 右上角菜单
2. **「添加到主屏幕」**

## 功能
- 🍽️ 浏览 100 道菜的菜单，6 个分类
- 🔍 搜索（支持中文、拼音、首字母）
- ❤️ 收藏 + 🌟 心愿单
- 🎲 随机推荐套餐
- 🛒 购物车 + 备注
- 📋 订单历史 + 再来一单
- ⭐ 吃完评价
- ⚙️ 管理后台（密码：gogo520）
- 📊 年度美食报告
- 💕 纪念日计数
- 🎉 节日彩蛋

## 数据存储
所有数据保存在浏览器的 **localStorage** 中，刷新不会丢失。

## 与微信小程序版的区别
| 对比 | 微信小程序 | 网页版 |
|------|-----------|--------|
| 费用 | 30 元/年认证费 | **0 元** |
| 使用方式 | 微信里打开 | **浏览器打开或桌面图标** |
| 通知 | Server酱推微信 | 暂无推送（可后续加） |
| 数据 | 云数据库 / 本地 | **本地 localStorage** |
| 持久性 | 云存储不丢 | 清浏览器缓存会丢 |
